# first-order absorption 1 compartment

first-order absorption 1 compartment

## Usage

``` r
OneCompOral(d)
```

## Arguments

- d:

  data, accepts a NONMEM style data frame for 1 subject with columns for
  TIME, AMT,MDV,DV, CL, V, KA & F1

## Value

Returns a dataframe with populated columns for A1, A2 and DV
