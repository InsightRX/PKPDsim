# First-order absorption- 2 compartment

First-order absorption- 2 compartment

## Usage

``` r
TwoCompOral(d)
```

## Arguments

- d:

  data, accepts a NONMEM style data frame for 1 subject with columns for
  TIME, AMT,MDV,DV, CL, V2, Q, V3, KA & F1

## Value

Returns a dataframe with populated columns for A1, A2, A3 and DV
