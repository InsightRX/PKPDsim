# Parse lagtime specified to main sim() function

Parse lagtime specified to main sim() function

## Usage

``` r
parse_lagtime(lagtime, ode, parameters)
```

## Arguments

- lagtime:

  either a value (numeric) or a parameter (character) or NULL.

- ode:

  function describing the ODE system

- parameters:

  model parameters

## Value

a vector of character or numeric values
