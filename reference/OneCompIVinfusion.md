# IV infusion- 1 compartment

IV infusion- 1 compartment

## Usage

``` r
OneCompIVinfusion(d)
```

## Arguments

- d:

  data, accepts a NONMEM style data frame for 1 subject with columns for
  TIME, AMT,MDV, RATE, RATEALL, DV, CL, V

## Value

Returns a dataframe with populated columns for A1, and DV
