# Create an event table

Create an event table

## Usage

``` r
create_event_table(
  regimen,
  t_max = NULL,
  t_obs = NULL,
  t_tte = NULL,
  t_init = 0,
  p,
  covariates,
  model = NULL,
  obs_type = NULL
)
```

## Arguments

- regimen:

  regimen

- t_max:

  t_max

- t_obs:

  t_obs

- t_tte:

  t_tte

- t_init:

  t_init

- p:

  parameters

- covariates:

  covariates

- model:

  model

- obs_type:

  observation type
