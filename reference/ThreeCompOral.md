# first-order absorption- 3 compartment

first-order absorption- 3 compartment

## Usage

``` r
ThreeCompOral(d)
```

## Arguments

- d:

  data, accepts a NONMEM style data frame for 1 subject with columns for
  TIME, AMT,MDV,DV, CL, V2, Q3, V3, Q4, V4, KA & F1

## Value

Returns a dataframe with populated columns for A1, A2, A3, A4 and DV
