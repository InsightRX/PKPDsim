# Description

`PKPDsim` is a library for numerical integration of ODE systems, in particular pharmacokinetic-pharmacodynamic (PK-PD) mixed-effects models.

## Installation

```
library(devtools)
install_github("InsightRX/PKPDsim")
library(PKPDsim)
```
